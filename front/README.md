# BaseIA - Frontend

Este es el frontend de BaseIA, construido con **React**, **Vite** y **Tailwind CSS v4**.

## 🚀 Requisitos

- Node.js (v18+)
- npm o yarn

## 🛠️ Instalación

```bash
npm install
```

## ⚙️ Configuración

Crea un archivo `.env` en la raíz del directorio `front/` copiando el archivo de ejemplo (si existe) o creando uno nuevo:

```env
VITE_API_URL=http://localhost:3000/api/v1
```

## 🏃 Ejecución

```bash
# servidor de desarrollo local
npm run dev

# construcción para producción
npm run build
```

La aplicación estará disponible por defecto en: [http://localhost:5173](http://localhost:5173).
